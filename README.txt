<GENERAL INSTRUCTIONS>

1. Run "RunMe.bat" to create a shortcut for the .exe file in the same directory.

2. Run "Shift Reduce Parser.exe".

3. Enter an expression using the following symbols:
	- i
	- +
	- *
	- (
	- )
	- $ (NOTE: all expressions must end with a "$"; the expression
		should only contain one instance of this character)

4. Click "Parse" to parse the expression.
	- Option: You may uncheck the "Highlight Cells" checkbox in the
		bottom right corner to stop the program from highlighting
		cells as the expression is parsed (this will speed up the
		process)

-----------------------------------------------------------------------------------
<RESET>

5. To parse another expression, you may enter your expression in the input
	box, click "Reset" to clear all of the lists, and then click "Parse" again
	to parse the new expression.

-----------------------------------------------------------------------------------
<NEXT STEP>

6. You may also click "Next Step" instead of "Parse" to run through the parsing
	a step at a time. NOTE: you may not alternate between "Parse" and "Next
	Step". "Reset" must be clicked before you can parse the expression in
	the other mode.
